# ⏰ Relógio Digital Minimalista

Um relógio digital minimalista desenvolvido com HTML, CSS e JavaScript puro. Com um design limpo em fundo preto e números brancos, exibe a hora em tempo real.

## 🌟 Funcionalidades

- ⏱️ Exibe hora, minutos e segundos em tempo real
- 🎨 Design minimalista e elegante
- 📱 Totalmente responsivo (funciona em mobile, tablet e desktop)
- ⚫ Tema escuro (fundo preto com texto branco)
- ⚡ Sem dependências externas
- 🔄 Atualização automática a cada segundo

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e responsividade
- **JavaScript** - Lógica e atualização em tempo real

## 📁 Estrutura do Projeto

```
relogio-digital/
├── index.html      # Arquivo principal HTML
├── style.css       # Estilos do relógio
├── script.js       # Lógica JavaScript
└── README.md       # Este arquivo
```

## 🎯 Como Funciona

1. O JavaScript captura a hora atual do sistema
2. Atualiza a exibição a cada segundo usando `setInterval()`
3. CSS fornece estilo visual atraente e responsivo
4. HTML estrutura o conteúdo semanticamente
